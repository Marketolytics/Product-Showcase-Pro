Premium WooCommerce + Elementor horizontal product showcase with enquiry system, PDF catalogue downloads, drag-and-drop form/popup builder, download tracking and an admin dashboard.

== Description ==

STC Product Showcase Pro turns a WooCommerce catalogue into a professional, B2B-style product showcase. It is built for industrial / catalogue websites where customers enquire about products instead of buying them online.

Features:

* Elementor widget "STC Product Showcase" with image-left / content-right horizontal cards.
* 8 ready-made layouts (Image Left, Image Right, Compact, Industrial Premium, Thermax Style, UKL Style, Minimal, Custom Builder).
* Query products by Category, Subcategory, Brand, SKU, Tags, Featured, Latest, Best Selling or Manual selection.
* Pagination, Load More and Infinite Scroll.
* Toggle every product element independently (image, name, SKU, brand, category, description, features, price, rating, tags, stock).
* Description word/character limit with Read More / Read Less.
* Features block sourced from WooCommerce meta, a custom field or an ACF field, shown as bullets, checkmarks or icons.
* "Enquire Now" popup that replaces Add To Cart, plus a "Download Catalogue" PDF button with download counter, file size and tracking.
* Per-product Catalogue PDF meta box (Media Library, uploaded URL, external URL, Google Drive or Dropbox).
* Advanced product meta: Technical Specifications, Features, Applications, Industries Served, Downloads, Certificates, Approvals.
* Drag-and-drop form / popup builder (text, email, phone, dropdown, radio, checkbox, textarea, hidden, date, file upload).
* Auto product data (ID, name, SKU, category, URL) passed to every enquiry.
* SMTP-compatible email notifications via wp_mail().
* Admin dashboard: Enquiries (view / delete / export CSV), Downloads report, Settings, Templates (import / export / global).
* Optimised queries for large catalogues (50k+ products), AJAX loading, lazy images.
* Security: nonces, capability checks, sanitisation and prepared statements throughout.

== Installation ==

1. Upload the `stc-product-showcase-pro` folder to `/wp-content/plugins/` (or install the ZIP via Plugins > Add New > Upload).
2. Activate the plugin.
3. Ensure WooCommerce and Elementor are active.
4. Edit a page with Elementor and drop in the "STC Product Showcase" widget.
